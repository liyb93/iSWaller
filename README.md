# iSWaller介绍

一款Mac端小巧壁纸应用，使用[pixabay](https://pixabay.com/)开放API。

功能：

\- 自动更换壁纸：15分钟、30分钟 、45分钟、1小时、2小时、1天等。

\- 搜索：支持模糊/精确搜索，快速找到喜爱的壁纸。

\- 快捷键：切换不喜欢的壁纸。

\- 预览壁纸质量切换：网络不好，切换图片预览质量。

\- 下载历史：记录使用的壁纸，防止喜爱的壁纸误删。
